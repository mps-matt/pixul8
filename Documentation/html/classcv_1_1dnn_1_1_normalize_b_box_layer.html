Detailed Description

- normalization layer.

Parameters
pNormalization factor. The most common p = 1 for - normalization or p = 2 for - normalization or a custom one.
epsParameter to prevent a division by zero.
across_spatialIf true, normalize an input across all non-batch dimensions. Otherwise normalize an every channel separately.

Across spatial:

Channel wise normalization:

Where x, y - spatial coordinates, c - channel.

An every sample in the batch is normalized separately. Optionally, output is scaled by the trained parameters.
